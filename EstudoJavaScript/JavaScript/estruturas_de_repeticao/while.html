While (enquanto)

A declaração while cria um laço que executa uma rotina especificada enquanto a condição de teste for verdadeira. A condição é avaliada antes da execução da rotina.

Sintaxe

while (condição) {
  rotina }

condição

Uma expressão avaliada antes de cada passagem através do laço. Se essa condição for avaliada com verdadeira, a rotina é executada. Quando a condição for avaliada como falsa, a execução continua na declaração depois do laço while.

rotina

Uma declaração que é executada quando a condição é avaliada como verdadeira. Para executar multiplas declarações dentro do laço, use uma declaração em bloco ({...}) para agrupar essas declarações.

exemplo

O código no loop será executado repetinamente, desde que uma variável (i) seja menor que 10:

while (i < 10) {
  text += "The number is " + i
  i++;
}
