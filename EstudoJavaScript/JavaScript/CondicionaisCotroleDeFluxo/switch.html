Declaração de troca de JavaScript

A instrução switch é usada para executar diferentes ações com base em diferentes condições.

Sintaxe

switch(expression) {
  case x:
    code block
    break;
  case y:
    code block
    break;
  default:
    code block
}

é assim que funciona:

  • A expressão switch é avaliada uma vez.
  • O valor da expressão é comparado com os valores de cada caso.
  • Se houver uma correspondêcia, o código do bloco associado será executado.
  • Se não houver correspondêcia o bloco de código padrão será executado.
