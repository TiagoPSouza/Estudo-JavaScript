Operadores de comparação

Os operadores de comparação são usados para testar true ou false, são usados em instruções logicas para determinar a igualdade ou diferença entre variáveis ou valores.

Os operandos podem ser numéricos, strings, lógicos ou objetos. Strings são comparadas com base em ordenação lexográfica utilizando valores Unicode. Na maoioria dos casos, se dois operandos são do mesmo tipo, o javascript tenta convertelos para um tipo apropriado. Isto geralmente se resulta na realização d uma comparação numérica. As únicas exceções a esta regra são os operadores (===) e o (!==), que realiza comparações de igualdade e desigualdade estritas.

operadores

  • Igual (==)
  • Diferente (!=)
  • Estritamente igual (===)
  • Estritamente diferente (!==)
  • Maior que (>)
  • maior ou igual (>=)
  • Menor que (<)
  • Menor ou igual (<=)

Nota:(=>) não é um operador, mas a notação para função de seta.
