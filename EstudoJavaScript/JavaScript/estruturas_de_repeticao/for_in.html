for...in (para em)

O laço for..in interage sobre propriedades enumeradas de um objeto, na ordem original da inserção. O laço pode ser executado para cada proriedade distinta do objeto.

Sintaxe

for (variavel in objeto) {
}

variável

Uma propriedade diferente do objeto é atribuida em cada iteração.

objeto

Objeto com as propriedades enumeradas.

Exemplo

const person = {fname:"John", Iname:"Doe", age:25};
let text = "";
for (let x in person) {
  text += person[x];
}
