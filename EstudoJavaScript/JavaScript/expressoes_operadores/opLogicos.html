Operadores Lógicos

Operadores lógicos são utilizados com valores booleanos (lógicos); neste caso, retornam um valor booleano. Entretanto os valores && e || na verdade retornam o valor de um dos operandos especificados, de forma de que esses operadores são utilizados com valores não booleanos, eles podem retornar um valor não booleano. Os operadores lógicos são descritos na tabela a seguir:

Operadores lógicos
Operador Utilização Descrição
lógico AND (&&) expr1 && expr2 Retorna expr1 caso possa ser convertido para falso, senão, retorna expr2. Assim, quando utilizado com valores booleanos, && retorna verdadeiro quando ambos operandos sejam verdadeiros, caso contrário, retorna falso.
lógico OR (||) expr1 || expr2 Retorna expr1 caso possa ser convertido para verdadeiro, senão, retorna expr2. Assim, quando utilizado com valores booleanos, || retorna verdadeiro quando ambos operandos sejam verdadeiros; se ambos forem falsos retona falso.
lógico NOT (!) !expr1 Retorna falso caso único operando possa ser convertido para verdadeiro; senão, retorna falso.
