for...of (para de)

O loop for...of percorre objetos iterativos (icluindo Array, Map, Set, o objeto arguments e assim por diante), chamando uma função personalizada com instruções a serem executadas para o valor de cada objeto distinto.

Sintaxe

for (variavel of iteravel) {
  declaração
}

variável

A cada interação, uma valor de propriedade diferente é atribuido a variável.

iteravel

Objeto cujo atributos serão iterados.

exemplo

Loop sobre um Array:

const cars = ["BMW", "Volvo", "Mini"];

let text = "";
for (let x of cars) {
  text += x;
}
