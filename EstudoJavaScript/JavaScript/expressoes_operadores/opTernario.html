Operador Condicional Ternário

O operador condicional (ternário) é o unico operador JavaScript que possui três operadores. Este operador é frequentemente usado com um atalho para instrução if

Sintaxe

condition ? expr1 : expr2

Parâmetros

condition

Uma expressão que é com true ou false.

expr1, expr2

Expressões com valores de qualquer tipo.

Descrição

Se condition é true, o operador retornará o valor de expr1; se não, ele retornará o valor de expr2.

exemplo

<input id="age" value="16">
<button onclick="myFunction()">Enviar</button>
<p id="demo"></p>

<script>
function myFunction() {
let age, voteable;
age = document.getElementById("age").value;
voteable = (age < 16) ? "muito jovem":"Velho o bastante";
document.getElementById("demo").innerHTML = voteable + "para votar.";
}
</script>
