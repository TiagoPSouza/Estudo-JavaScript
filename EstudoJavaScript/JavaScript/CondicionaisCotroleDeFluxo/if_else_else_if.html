JavaScript if, else e else if

As instruções condicionais são usadas para executar doferentes ações com base em diferentes condições.

Declarações condicionais

  • Use if para especificar um bloco de código a ser executado, se uma condição especificada for verdadeira.
  • Use else para especifiar um bloco de código a ser executado, se a mesma condição for falsa.
  • Use else if para especificar uma nova condição a ser testada, se a primeira condição for falsa.

Exemplo de if else

Se a hora for menor que 18, crie uma saudação "Good day", caso contrário, "Good evening":

if (hour < 18) {
greeting = "good day";
} else {
greeting = "Good evening";
}

Exemplo de if, else if, else

Se a hora for menor que 10:00, crie uma saudação "Good morning", se não, mas a hora for menor que 20:00, crie a saudação "Good day", caso contrário, uma saudação "Good evening":

if (time < 10) {
greeting = "Good morning!!!";
} else if (time < 20) {
greeting = "Good day!!!";
} else {
greeting = "Good evening!!!";
}
