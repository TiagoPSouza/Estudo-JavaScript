For

A instrução for cria um loop que consiste em três expressões opcionais, dentro de parênteses e separadas por ponto e vígula (;), seguida por uma declaração ou uma sequência de declarações executadas em sequência.

Sintaxe

for ([inicialização]; [condição]; [expressão final]) {
  declaração }

inicialização

Uma expressão (incluindo expressões de atribuição) ou declarções variáveis. Geralmente usada para iniciar o contador de variáveis. Esta expressão pode, opcionalmente, declarar novas variáveis com a palavra chave var. Essas variáveis não são locais no loop, isto é, elas estão no mesmo escopo que o loop for está. Variáveis com let são locais para a declaração.

condição

Uma expressão para ser avaliada antes de cada interação do loop. Se está expressão for avaliada para true, a declaração sera executada. Este teste de condição é opcional. Se omitido, a codição será sempre avaliada como true. Se a expressão for avaliada como false, a execução irá para a pimeira expressão após a construção do loop for.

expressão final

Uma expressão que será avaliada no final de cada interação no loop. Isso ocorre antes da próxima avaliação de condição. Geralmente usada para incremetar ou atualizar a variáveis do contador.

declaração

Uma declaração que é executada quando a condição for true. Para excutar múltiplas condições dentro do loop, use uma instrução de bloco ({...}) para agrupar as condições. Para não executar declarações dentro do loop, use uma instrução vazia (;).

exemplo

let text = ""
for (let i = 0; i < 5; i++) {
  text += "O número é " + i;
}
